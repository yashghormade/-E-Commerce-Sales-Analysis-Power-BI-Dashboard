📊 E-Commerce Sales Analysis & Power BI Dashboard
📌 Project Overview

This project focuses on Exploratory Data Analysis (EDA) of real-world E-Commerce sales data using Python and Power BI. The analysis was performed on 9,800+ Superstore orders to uncover valuable business insights related to sales trends, customer behavior, regional performance, and product analysis.

The project aims to analyze:

📈 Sales Trends
🛒 Product Performance
🌍 Regional Analysis
👥 Customer Segments
🚚 Shipping Analysis
📊 Business Growth Patterns

An interactive Power BI dashboard was built to visualize the insights effectively.
